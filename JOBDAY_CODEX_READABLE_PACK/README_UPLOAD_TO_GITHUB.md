# 가장 쉬운 설치 순서

## 방법 1: GitHub 웹에서 올리기
1. 이 폴더의 파일을 컴퓨터에 내려받습니다.
2. GitHub에서 JOBDAY 저장소를 엽니다.
3. `Add file` → `Upload files`를 누릅니다.
4. 이 폴더 안의 `AGENTS.md`, `CODEX_START_HERE.md`, `PROMPT_00_AUDIT_AND_PLAN.txt`, `docs` 폴더를 저장소 최상단에 올립니다.
5. `Commit changes`를 누릅니다.
6. Codex에서 JOBDAY 저장소와 최신 브랜치를 선택합니다.
7. `PROMPT_00_AUDIT_AND_PLAN.txt` 내용을 Codex 입력창에 붙여넣습니다.

## 중요한 점
- ZIP 파일 자체를 저장소에 올리는 것만으로는 안 됩니다. 압축을 풀어 내부 파일이 실제 경로에 존재해야 합니다.
- `AGENTS.md`는 저장소 최상단에 있어야 전체 프로젝트에 적용됩니다.
- 기존 `docs` 파일과 이름이 겹치면 먼저 백업하거나 Codex가 충돌을 감사하게 하십시오.
